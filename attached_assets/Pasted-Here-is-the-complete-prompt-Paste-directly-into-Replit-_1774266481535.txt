### Packing Charges Management System — Takeaway & Delivery

This is a continuation of the existing restaurant SaaS project in Replit. The following are already built: multi-tenant architecture, POS, QR ordering, billing and payments, multi-outlet pricing, delivery management, and outlet settings.

Now build a **fully functional Packing Charges Management System** that is completely controlled by the restaurant owner — packing charges are hidden or shown based on owner configuration, applied automatically to takeaway and delivery orders, with full flexibility on how charges are calculated and displayed to customers.

***

#### 1. Core concept

```
OWNER configures packing charges in outlet settings
        ↓
System auto-detects order type:
  DINE_IN    → No packing charge (never)
  TAKEAWAY   → Apply if enabled by owner
  DELIVERY   → Apply if enabled by owner
        ↓
Charge applied automatically at billing:
  → Fixed amount per order
  → Fixed amount per item
  → Percentage of subtotal
  → Per item category (different charge
     for different food types)
        ↓
Shown clearly on:
  → POS cart
  → QR order summary
  → Bill and receipt
  → Online delivery platforms
        ↓
Owner can:
  → Enable / disable completely
  → Set different rates for takeaway vs delivery
  → Exempt specific items or categories
  → Set tax on packing charge separately
```

#### 2. Packing charge configuration schema

```sql
-- Packing charge settings per outlet
CREATE TABLE outlet_packing_settings (
  id                      UUID PRIMARY KEY,
  tenant_id               UUID NOT NULL,
  outlet_id               UUID NOT NULL UNIQUE,

  -- Master switches (owner only)
  takeaway_charge_enabled BOOLEAN DEFAULT false,
  delivery_charge_enabled BOOLEAN DEFAULT false,

  -- Charge type
  charge_type             VARCHAR(20) DEFAULT 'FIXED_PER_ORDER',
    -- 'FIXED_PER_ORDER'   → flat charge per order
    -- 'FIXED_PER_ITEM'    → charge per item in order
    -- 'PERCENTAGE'        → % of order subtotal
    -- 'PER_CATEGORY'      → different charge by food type

  -- Rates
  takeaway_charge_amount  DECIMAL(10,2) DEFAULT 0,
    -- used for FIXED_PER_ORDER or PERCENTAGE
  delivery_charge_amount  DECIMAL(10,2) DEFAULT 0,
    -- used for FIXED_PER_ORDER or PERCENTAGE

  -- Per item rates (used for FIXED_PER_ITEM)
  takeaway_per_item       DECIMAL(10,2) DEFAULT 0,
  delivery_per_item       DECIMAL(10,2) DEFAULT 0,

  -- Maximum cap (optional)
  max_charge_per_order    DECIMAL(10,2),
    -- NULL = no cap

  -- Tax on packing charge
  packing_charge_taxable  BOOLEAN DEFAULT false,
  packing_charge_tax_pct  DECIMAL(5,2) DEFAULT 0,

  -- Display settings
  show_on_receipt         BOOLEAN DEFAULT true,
  charge_label            VARCHAR(100)
    DEFAULT 'Packing Charge',
    -- owner can rename: 'Packaging', 'Box Charge', etc.

  -- Currency (from outlet)
  currency_code           VARCHAR(10) DEFAULT 'INR',
  currency_symbol         VARCHAR(5) DEFAULT '₹',

  updated_by              UUID,
  updated_at              TIMESTAMP DEFAULT NOW(),
  created_at              TIMESTAMP DEFAULT NOW()
);

-- Per category packing charge rates
-- Used when charge_type = 'PER_CATEGORY'
CREATE TABLE packing_charge_categories (
  id                      UUID PRIMARY KEY,
  tenant_id               UUID NOT NULL,
  outlet_id               UUID NOT NULL,
  category_name           VARCHAR(100) NOT NULL,
    -- 'Liquids', 'Dry Items', 'Fragile', 'Combos'
  takeaway_charge         DECIMAL(10,2) DEFAULT 0,
  delivery_charge         DECIMAL(10,2) DEFAULT 0,
  applies_to_categories   JSONB,
    -- menu category ids this rate applies to
  created_at              TIMESTAMP DEFAULT NOW()
);

-- Per item packing charge exemptions
CREATE TABLE packing_charge_exemptions (
  id                      UUID PRIMARY KEY,
  tenant_id               UUID NOT NULL,
  outlet_id               UUID NOT NULL,
  exemption_type          VARCHAR(20) NOT NULL,
    -- 'MENU_ITEM'     → specific item exempt
    -- 'CATEGORY'      → entire category exempt
  reference_id            UUID NOT NULL,
    -- menu_item_id or category_id
  reference_name          VARCHAR(255),
  reason                  TEXT,
  created_at              TIMESTAMP DEFAULT NOW()
);

-- Packing charge log per bill
CREATE TABLE bill_packing_charges (
  id                      UUID PRIMARY KEY,
  tenant_id               UUID NOT NULL,
  outlet_id               UUID NOT NULL,
  bill_id                 UUID NOT NULL,
  order_id                UUID NOT NULL,
  order_type              VARCHAR(20) NOT NULL,
    -- 'TAKEAWAY', 'DELIVERY'
  charge_type             VARCHAR(20) NOT NULL,
  charge_amount           DECIMAL(10,2) NOT NULL,
  tax_amount              DECIMAL(10,2) DEFAULT 0,
  total_amount            DECIMAL(10,2) NOT NULL,
  item_count              INT,
  breakdown               JSONB,
    -- per item breakdown if FIXED_PER_ITEM
  created_at              TIMESTAMP DEFAULT NOW()
);
```

#### 4. Charge type configuration screens

##### Option A — Fixed per order (simplest)
```
Takeaway: ₹ [25] flat per order
Delivery: ₹ [40] flat per order
```

##### Option B — Fixed per item
```
Takeaway: ₹ [5 ] per item in order
Delivery: ₹ [8 ] per item in order

Example:
  Order has 4 items → delivery charge = 4 × ₹8 = ₹32
  Maximum cap: ₹100 → charge never exceeds ₹100
```

##### Option C — Percentage of subtotal
```
Takeaway: [3  ]% of subtotal
Delivery: [5  ]% of subtotal

Example:
  Subtotal ₹800 → delivery charge = 5% = ₹40
  Maximum cap: ₹80
```

##### Option D — Per food category
```

#### 5. Packing charge calculation engine

```typescript
class PackingChargeService {

  async calculatePackingCharge(
    outletId: string,
    orderType: 'TAKEAWAY' | 'DELIVERY',
    orderItems: OrderItem[]
  ): Promise<PackingChargeResult> {

    // Get outlet settings
    const settings = await getPackingSettings(outletId);

    // Check if enabled for this order type
    if (orderType === 'TAKEAWAY'
        && !settings.takeaway_charge_enabled) {
      return { amount: 0, applicable: false };
    }

    if (orderType === 'DELIVERY'
        && !settings.delivery_charge_enabled) {
      return { amount: 0, applicable: false };
    }

    // Filter out exempted items
    const chargeableItems = await this.filterExemptions(
      outletId, orderItems
    );

    if (chargeableItems.length === 0) {
      return { amount: 0, applicable: false,
               reason: 'All items exempt' };
    }

    // Calculate based on charge type
    let charge = 0;
    let breakdown = [];

    switch (settings.charge_type) {

      case 'FIXED_PER_ORDER':
        charge = orderType === 'TAKEAWAY'
          ? settings.takeaway_charge_amount
          : settings.delivery_charge_amount;
        break;

      case 'FIXED_PER_ITEM':
        const ratePerItem = orderType === 'TAKEAWAY'
          ? settings.takeaway_per_item
          : settings.delivery_per_item;

        for (const item of chargeableItems) {
          const itemCharge = item.quantity * ratePerItem;
          charge += itemCharge;
          breakdown.push({
            item: item.name,
            qty: item.quantity,
            rate: ratePerItem,
            amount: itemCharge
          });
        }
        break;

      case 'PERCENTAGE':
        const subtotal = chargeableItems.reduce(
          (sum, item) => sum + item.lineTotal, 0
        );
        const pct = orderType === 'TAKEAWAY'
          ? settings.takeaway_charge_amount
          : settings.delivery_charge_amount;
        charge = (subtotal * pct) / 100;
        break;

      case 'PER_CATEGORY':
        const categoryRates = await getCategoryRates(outletId);
        for (const item of chargeableItems) {
          const rate = this.findCategoryRate(
            item, categoryRates, orderType
          );
          const itemCharge = item.quantity * rate;
          charge += itemCharge;
          breakdown.push({
            item: item.name,
            category: item.category,
            qty: item.quantity,
            rate,
            amount: itemCharge
          });
        }
        break;
    }

    // Apply maximum cap
    if (settings.max_charge_per_order
        && charge > settings.max_charge_per_order) {
      charge = settings.max_charge_per_order;
    }

    // Calculate tax if applicable
    const taxAmount = settings.packing_charge_taxable
      ? (charge * settings.packing_charge_tax_pct) / 100
      : 0;

    return {
      applicable: true,
      chargeType: settings.charge_type,
      amount: charge,
      taxAmount,
      totalAmount: charge + taxAmount,
      breakdown,
      label: settings.charge_label
    };
  }
}
```
