### Complete Parking Management System — Valet + Slots + Charges + QR Integration

This is a continuation of the existing restaurant SaaS project in Replit. The following are already built: multi-tenant architecture, POS, table management, floor plan, QR customer requests, billing and payments, outlet settings, audible alert system, and printer integration.

Now build a **complete, fully integrated Parking Management System** covering everything in one module — parking slot layout with visual board, valet operations, charge management, QR vehicle retrieval, and owner controls — all working together seamlessly.

***

#### 1. Complete operational flow

```
OWNER SETS UP PARKING (one time):
  → Configure zones and slots
  → Set charge mode and rates
  → Set validation rules
  → Configure display to customers
          ↓
CUSTOMER ARRIVES:
  → Valet receives vehicle
  → Records entry on mobile device
  → Assigns parking slot
  → Prints 2-copy ticket
  → Customer gets ticket copy
          ↓
CUSTOMER DINING:
  → Valet ticket linked to table
  → Parking timer running
  → Staff sees slot occupied on board
          ↓
BILL GENERATED:
  → Parking charge auto-calculated
  → Added to restaurant bill
  → Validation applied if bill qualifies
  → Customer pays everything together
          ↓
CUSTOMER REQUESTS VEHICLE:
  → Via QR on table after payment
  → Via waiter or cashier
  → Valet notified instantly
  → Retrieves vehicle from slot
  → Slot freed in system
          ↓
CUSTOMER EXITS:
  → Vehicle handed over
  → Ticket closed
  → Slot available again
```
