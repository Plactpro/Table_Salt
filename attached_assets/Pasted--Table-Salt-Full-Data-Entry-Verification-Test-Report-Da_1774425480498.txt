### Bugs Found (3 Total)

#### 🔴 BUG #1 — Add Inventory Item: Error 500
- **Module:** Inventory → Add Item
- **Error:** `Error 500: invalid input syntax for type numeric: '""'`
- **Trigger:** Clicking "Add Item" and submitting the form
- **Impact:** Cannot add any new inventory item — this blocks stock tracking for new ingredients
- **Likely Cause:** A numeric field (possibly SKU, category ID, or supplier ID) is passing an empty string instead of null or a valid number

#### 🔴 BUG #2 — Add Zone: Button Not Working
- **Module:** Tables → Zones → "+ Add"
- **Error:** No toast, no new zone created, list stays at 7 zones
- **Trigger:** Entering zone name + selecting color + clicking "+ Add" button
- **Impact:** Cannot create new seating zones
- **Likely Cause:** Button's onClick handler may not be wired correctly, or API call failing silently

#### 🟡 BUG #3 — Tables Page: Transient React Error #185
- **Module:** Tables (Zones tab navigation)
- **Error:** "Could not load tables — Minified React error #185"
- **Trigger:** Navigating from Online Orders → Tables in quick succession
- **Impact:** Low — page recovers after clicking "Retry" button
- **Likely Cause:** Race condition on component mounting / async data load
