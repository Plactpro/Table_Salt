**2. Database Schema for Combos**
```sql
CREATE TABLE combo_offers (
  id UUID PRIMARY KEY,
  tenant_id UUID NOT NULL,
  name VARCHAR(255) NOT NULL,
  description TEXT,
  combo_price DECIMAL(10,2) NOT NULL,
  individual_total DECIMAL(10,2) NOT NULL,
  savings_percentage DECIMAL(5,2) NOT NULL,
  image_url TEXT,
  main_items JSONB[], -- Array of {item_id, quantity}
  side_items JSONB[],  -- Array of {item_id, quantity}
  addons JSONB[],      -- Array of {item_id, quantity}
  validity_start DATE,
  validity_end DATE,
  time_slots JSONB,    -- ["breakfast", "lunch", "dinner"]
  outlets JSONB[],     -- outlet_ids where available
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW()
);
```

**6. Real-time % Savings Calculation Formula**
```javascript
const calculateSavings = (comboPrice, items) => {
  const individualTotal = items.reduce((sum, item) => 
    sum + (item.price * item.quantity), 0
  );
  const savingsAmount = individualTotal - comboPrice;
  const savingsPercent = ((savingsAmount / individualTotal) * 100).toFixed(1);
  
  return {
    individualTotal,
    savingsAmount: savingsAmount.toFixed(2),
    savingsPercent,
    isValid: savingsPercent > 0 && savingsPercent <= 50 // Business rule
  };
};
```
