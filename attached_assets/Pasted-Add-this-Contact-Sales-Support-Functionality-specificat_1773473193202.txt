## **Dual Contact Functionality: Sales (Billing/Subscription) + Support (Top Right)**

### **1. Contact Sales – Billing & Subscription Pages Only**

**Strategic Placement:**
```
Billing Page → Fixed bottom-right "Contact Sales" button
Subscription Page → Fixed bottom-right "Contact Sales" button
ALL OTHER PAGES → Contact Sales HIDDEN
```

**Visual Design:**
```
Position: fixed; bottom: 30px; right: 30px; z-index: 1000;
Size: 60px height, rounded pill shape
Colors: Gold gradient background (#FFD700 → #FFA500), white bold text
Icon: Phone + Envelope (left of text)
Text: "Contact Sales"
Pulse animation (subtle glow) to draw attention on Billing/Subscription pages
Hover: Scale 1.05 + brighter gold
```

**Trigger Behavior:**
```javascript
// Show only on specific pages
const salesPages = ['/billing', '/subscription', '/pricing'];
if (salesPages.includes(currentPage)) {
  showContactSalesButton();
}
```

### **2. Contact Support – Top Right Corner (Global)**

**Global Header Placement:**
```
Position: fixed; top: 20px; right: 20px; z-index: 1001;
Always visible on ALL pages (including mobile)
```

**Visual Design:**
```
Size: 48px square → expands to 320px on hover/click
Colors: Teal theme (#00A8CC background), white icons/text
Icon: Headset + Chat bubble
Tooltip: "Need help? Contact Support"
Click → Instant chat widget or support form
```

**Two Interaction Modes:**
```
A. QUICK SUPPORT (80% use case)
  Click → Opens instant support overlay (chat-like)
  Pre-filled: Current page, user timezone, currency, subscription tier
  
B. FULL SUPPORT FORM (backup)
  Email + phone + issue type dropdown
```

### **3. Contact Sales Form (Billing/Subscription Trigger)**

**Enhanced Fields for Sales Context:**
```
REQUIRED:
├── Full Name
├── Business Name / Restaurant Name  
├── Business Type (dropdown from earlier list)
├── Number of Outlets
├── Country / City
├── Email Address
├── Phone / WhatsApp

SALES-SPECIFIC:
├── Current Subscription Tier (auto-detected)
├── Interested Plans (multi-select: Basic→Enterprise)
├── "Request Demo" checkbox
├── "Number of Locations" slider (1-50+)
├── Estimated Monthly Revenue range
└── Custom Requirements (textarea)
```

**Auto-populated Context:**
```javascript
// Pre-fill based on user session
formData = {
  timezone: getUserTimeZone(),  // Asia/Dubai
  currency: getUserCurrency(),  // AED
  subscription: getUserTier(),  // Premium
  businessType: getBusinessType(), // QSR
  currentPage: 'billing'  // Context
}
```

**Email Routing:**
```
TO: sales@restaurantapp.com (configurable)
CC: sales-manager@restaurantapp.com
Subject: "Sales Inquiry - [Business Name] - [Tier]"
```

### **4. Contact Support Form (Top Right Trigger)**

**Support-Specific Fields:**
```
QUICK MODE (Minimal):
├── Issue Type (dropdown)
│   ├── POS not loading
│   ├── Billing issue  
│   ├── Menu sync problem
│   ├── Staff scheduling
│   ├── Reservation conflict
│   ├── Other
├── Short Description (100 chars max)
└── Submit → Instant response

FULL MODE (Detailed):
├── Same as Sales + Technical Details
├── Screenshots (file upload)
├── Browser/Device info (auto-capture)
├── Recent Actions (session log snippet)
└── Urgency (Low/Medium/High/Critical)
```

**Email Routing:**
```
TO: support@restaurantapp.com (configurable)
CC: tier1-support@restaurantapp.com  
Subject: "[URGENCY] Support: [Issue Type] - [Business Name]"
```

### **5. Shared Email Infrastructure**

**Centralized Email Configuration:**
```javascript
const CONTACT_CONFIG = {
  sales: {
    email: 'sales@restaurantapp.com',
    cc: ['sales-manager@restaurantapp.com'],
    subjectPrefix: 'Sales Inquiry'
  },
  support: {
    email: 'support@restaurantapp.com', 
    cc: ['tier1-support@restaurantapp.com'],
    subjectPrefix: 'Support Request'
  }
};
```

**Email Template Generator:**
```javascript
function generateEmailTemplate(formData, type) {
  return `
  FROM: ${formData.email}
  BUSINESS: ${formData.businessName}
  TIER: ${formData.subscription}
  ISSUE: ${formData.issueType}
  MESSAGE: ${formData.message}
  TIMEZONE: ${formData.timezone}
  CURRENCY: ${formData.currency}
  PAGE: ${formData.currentPage}
  BROWSER: ${navigator.userAgent}
  `;
}
```
