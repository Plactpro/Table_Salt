### Cash Machine Integration System

This is a continuation of the existing restaurant SaaS project in Replit. The following are already built: multi-tenant architecture, POS, billing and payments, multi-outlet pricing, subscription system, and outlet settings.

Now build a **fully functional Cash Machine Integration System** that connects the POS to a physical cash drawer, handles cash transactions with exact change calculation, supports multiple currencies based on outlet settings, manages cash float, and maintains a complete cash audit trail — working seamlessly with the existing billing and payment module.

***

#### 1. Core concept and flow

```
CASHIER opens shift
        ↓
Opens cash drawer → counts opening float
→ Records opening amount in system
        ↓
Customer pays bill
        ↓
Cashier enters amount tendered
→ System calculates exact change
→ Cash drawer opens automatically
→ Cashier gives change
→ Transaction recorded
        ↓
Throughout shift:
→ Cash drawer opens only for cash transactions
→ Every open logged with reason
→ Running cash balance tracked in real time
        ↓
Cashier closes shift
→ Counts physical cash in drawer
→ System shows expected amount
→ Variance flagged and explained
→ Manager approves close
→ Cash handover recorded
```

#### 6. Change calculation engine

```typescript
class ChangeCalculationService {

  calculateChange(
    amountDue: number,
    tendered: number,
    currency: CurrencyConfig
  ): ChangeResult {

    // Step 1: Apply rounding rule
    const roundedDue = this.applyRounding(
      amountDue,
      currency.rounding
    );

    // Step 2: Calculate change
    const change = tendered - roundedDue;

    if (change < 0) {
      return { valid: false, error: 'Insufficient amount' };
    }

    // Step 3: Break change into denominations
    const breakdown = this.denominationBreakdown(
      change,
      currency
    );

    return {
      valid: true,
      amountDue: roundedDue,
      tendered,
      change,
      breakdown
    };
  }
